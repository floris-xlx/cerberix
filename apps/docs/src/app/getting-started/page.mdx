## Getting Started

1. Start services:

```bash
docker compose up -d
```

2. Configure env vars (see repo README).

3. Run dev:

```bash
pnpm dev
```

4. Create a project in the dashboard and copy your API keys.

5. Publish an event:

```bash
curl -X POST http://localhost:4001/api/v1/events \
  -H "X-Project-Key: <public>" \
  -H "Authorization: Bearer <secret>" \
  -H "Content-Type: application/json" \
  -d '{"type":"user.created","payload":{"id":"u_123"}}'
```
